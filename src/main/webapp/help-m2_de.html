Jenkins kann Reports von PMD's Copy Paste Detector (CPD) einlesen und darstellen. Wenn dies aktiviert wird und der Build das Maven Goal pmd:cpd ausführt, dann analysiert Jenkins nach jedem Build die erzeugten CPD Dateien und stellt deren Informationen in verschiedenen Ansichten dar: Trend Anzeige, Projekt Statistik und Package Statistik. Zudem zeigt Jenkins die jeweiligen Warnungen direkt in der betroffen Datei an.

Das Plug-in wird nicht aufgerufen, wenn der Build fehlgeschlagen ist, sondern nur für stabile bzw. instabile Builds (mit fehlgeschlagenen Tests).
